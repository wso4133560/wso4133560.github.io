    ssh操作

    linux命令
    Created At : 2023-11-24 21:35
    Views 👀 :
    1. ssh常规命令
    2. ssh带端口
    3. 配置文件路径
    4. linux下sshfs挂载文件命令

    ssh常规命令

    ssh [账户]@[密码]

    ssh带端口

    ssh [账户]@[密码] -p [端口]

    配置文件路径

    /etc/ssh/sshd_config

    linux下sshfs挂载文件命令

    linux上映射远端机器的硬盘到本地
    sudo sshfs -o cache=yes,allow_other [账号]@[ip]:[远端的文件夹] [本地的文件夹]

    取消挂载
    sudo fusermount -u [文件夹]
